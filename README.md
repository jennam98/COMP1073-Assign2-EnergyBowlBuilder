# COMP1073-Assign2-EnergyBowlBuilder

## Project Description
This is a web-based application that allows customers to build and order a customizable energy bowl. Users can choose a base, fruits, toppings, protein add-ons, sweeteners, size, and special instructions. After submitting the form, the application creates an EnergyBowl object and displays a complete bowl summary.

## Features
- Interactive customization form
- Object-oriented JavaScript using EnergyBowl class
- Dynamic order summary
- Basic input validation
- Price calculation
- Responsive front-end design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git and GitHub
